## 第二章：认知层 — TA 怎么想

> 这是腊月小九解读世界的方式，是一切内容的底层逻辑

### 核心信念（5条）

**信念1：做饭的动力来自孩子，不是食物本身**
> "只有孩子回家才想好好做饭"——这不是一句话，是整个账号的存在逻辑。
> 出现在：《只有孩子回家才想好好做饭！》《投喂高中住宿生》《什么是波奇饭》

**信念2：投喂是妈妈表达爱的语言**
> 孩子住校→做冰面包让他带走；孩子返校→前一天做轻食早餐；孩子假期回家→院子里第一场烧烤。
> 爱不说出口，全在食物里。
> 出现在：TOP1《烤肉》、TOP2《冰面包》、TOP3《返校Vlog》

**信念3：复刻是一种较劲，不是复制**
> "就复刻方便面这手艺，我还真没服过谁"——带着自信的挑战感，不是谦虚的学习感。
> 出现在：《复刻亲妈版方便面》《最近超火的刘什么祥麻辣》《复刻美食系列之冠军猪扒饭》

**信念4：孩子逐渐长大、"高冷"了，是妈妈的甜蜜烦恼**
> "不知道这份热情能不能配得上逐渐高冷的儿子"——这种温柔的自嘲是粉丝共情的核心。
> 出现在：TOP8《神兽回来啦》、TOP3《返校Vlog》评论区

**信念5：家里的食物永远比外面好，但不需要证明这一点**
> 不主动对比外卖/餐厅，只展示自家做的，让画面说话。
> 贯穿全部30条内容。

### 认知框架

腊月小九用**"情感时间轴"视角解读每道菜**：不问"这菜好不好吃"，而问"这道菜对应儿子生命里的哪个时刻"。

- 儿子两周才回来一次 → 第一场烧烤要安排上（时间稀缺感）
- 寒假住腻了、天天盼着回学校 → 轻食早餐是最后的温柔（分离叙事）
- 住校早餐吃不好 → 做冰面包带走（隔空投喂感）

这个框架让同一道食物有了叙事重量，这是她账号溢价的核心。

### 观点张力（矛盾点）

**矛盾**：「儿子是做饭最大动力」vs「儿子不在时也照常更新」

- 儿子在家时：大鱼大肉、烤肉、宴席级别
- 儿子返校后：高能量的一天、复刻系列、自己的碳水快乐
- 如何理解：儿子是情感触发器，但并非内容唯一素材；账号情感核心是"妈妈的热情"，不是"儿子的存在"

### 价值立场

高频有效词：**投喂、住宿、复刻、回家、返校、儿子、挑战**

> TA的内容底色是：**一个普通妈妈用食物丈量孩子成长的时间**

## 第三章：策略层 — TA 怎么运营

### 发布节奏

- 约27天/条（月更为主，极低频）
- 全视频，无图文（30/30）
- 每条独立成篇，无强系列标题，但有隐性系列（儿子系列、复刻系列）

### 内容系列结构

| 系列 | 触发条件 | 特征 | 占比估算 |
|------|---------|------|--------|
| 儿子归家大餐 | 儿子回家/放假 | 情感密度最高，数据最好 | ~30% |
| 投喂住宿生 | 儿子返校/住校 | 操作展示+情感结尾 | ~20% |
| 复刻挑战 | 网络热点食物 | 带较劲感，竞争性标题 | ~20% |
| 日常美食Vlog | 无特定触发 | 自己的口腹之欲 | ~30% |

### 蹭热点方式

- 蹭**食物热点**（刘文祥麻辣、波奇饭），不蹭情感热点或社会事件
- 热点策略：**复刻验证**型——"我做出来了，尝尝是不是这个味"
- 不追热点综艺/明星，只追食物本身的爆红

### 运营习惯

- 全部@王皓琦（儿子账号），形成母子互动话题场
- 儿子在评论区互动，带来二次流量（王皓琦的回复本身就是高赞评论）
- 标签策略：每条2-5个，精准垂类标签为主（#烤肉 #复刻美食），无通用堆砌
- CTA使用率仅13.3%（极低），基本靠内容自然留存

### IF-THEN 触发规则

```
IF 儿子放假回家
  → 大宴+烧烤+满桌硬菜，标题用感叹型或情感型

IF 儿子返校
  → 投喂类（可带走零食/面包），标题用"返校""轻食""投喂住宿生"关键词

IF 网络出现食物热点
  → 复刻挑战，标题用"最近超火的XXX""我做出来了""复刻XX"

IF 节假日
  → 院子烧烤/家宴，强调"只有这天才会做"的仪式感

IF 有儿子出镜/提及
  → 必须@王皓琦，正文结尾留互动钩子
```

## 第四章：内容层 — TA 怎么写

### 标题公式（5个）

**公式1：疑问型（21.4%）— 用陌生感引入**
> 格式：「XX是什么？+ 情境说明」
> 示例：「什么是波奇饭？高中生开学前最后一天」
> 适用：复刻新食物、介绍小众菜品

**公式2：感叹型（10.7%）— 情感宣言**
> 格式：「情感判断句 + ！」
> 示例：「只有孩子回家才想好好做饭！」
> 适用：儿子归家、重大时刻、情绪峰值时刻

**公式3：故事型（10.7%）— 日常矛盾感**
> 格式：「反常识的日常观察」
> 示例：「儿子明明都住校去了，我每天怎么还这么忙」
> 适用：日常Vlog、自嘲、展示忙碌日常

**公式4：挑战宣言型（~10%）— 带较劲感**
> 格式：「就XX这XX，我还真没服过谁」或「XX，能有多好吃」
> 示例：「就复刻方便面这手艺，我还真没服过谁」
> 适用：复刻系列、挑战类内容

**公式5：程度夸张型（~7%）— 量化诱惑**
> 格式：「这样的XX，一人吃N口不过分吧」
> 示例：「这样的小碗粉，一人吃五六七八碗不过分吧」
> 适用：食物展示、美食诱惑类

### 开头模板（3个）

**开头A：时间触发型**
> 「[孩子状态]，[所以做了什么]，[情绪表达]」
> 示例：「这小子两周才回来一次，这不，2026年院子里第一场烧烤就给他安排上了！」

**开头B：担忧投喂型**
> 「[担忧孩子某方面]+[所以准备了什么]」
> 示例：「儿子高中住校怕他早餐吃不好，给他做几个可以冷吃的冰面包当早餐吧！」

**开头C：挑战宣言型**
> 「我[动作]+[结果宣言/邀请对方验证]」
> 示例：「我做出来了，尝尝是不是这个味」
> 示例：「一家子碳水脑袋，我敢说看完视频都得流口水」

### 结尾与CTA

- **基本不写CTA**（使用率13.3%）
- 偶尔用：「收藏备用」——仅在操作干货类内容中出现
- 主要靠**正文末尾的情绪留白**自然留存：「哎～」「就是这么定了」
- @王皓琦 是最常见的"软CTA"——触发儿子出来互动

### 正文风格规格

| 维度 | 规格 |
|------|------|
| 字数 | 约50字（极简，视频内容为主，文字辅助） |
| Emoji | 0%（禁止使用Emoji） |
| 列表 | 0%（禁止用• 或1. 2. 3. 列举） |
| 小标题 | 0%（禁止分段小标题） |
| 语气 | 口语化，偶尔自嘲，不说教 |
| 人称 | 第一人称"我"为主，儿子称"这小子" |

### 标签策略

- 每条2-5个
- 垂类精准标签（#烤肉 #复刻美食 #返校）
- 偶尔加场景标签（#美食日常 #美食VLOG）
- 不堆砌热门通用标签（#美食 #家常菜 等基本不出现）

## 第五章：创作禁区

> 以下是腊月小九风格的绝对禁区，违反即破坏人设

**禁区1：教程化**
> 不要写步骤式菜谱。腊月小九从不写"第一步...第二步..."。
> 她的视频是Vlog，不是教程。文字只是情感注脚，不是操作说明。

**禁区2：使用Emoji**
> 全30条笔记0% Emoji使用率。一个😊都不要出现。
> 这是她区别于其他美食博主最强的辨识度之一。

**禁区3：精致化/完美化**
> 不要写"颜值满分的轻食早餐"这种话。她的内容是真实家庭感，不是网红餐厅感。
> 允许有点乱、有点挤、有点"这不就是普通家里么"的感觉。

**禁区4：无情感钩子的纯食物内容**
> 纯粹写"今天做了什么菜，食材是XXX，步骤是XXX"——这不是腊月小九。
> 每道菜背后必须有人（儿子、家人、自己的口腹之欲）。

**禁区5：说教或建议**
> 不要写"建议大家也这样做""这个配方值得收藏"之类的句子。
> 她从不给读者建议，只分享自己正在做的事。

## 第六章：对比示例

### 示例1：儿子归家场景

❌ 普通写法：
> 今天儿子放假回家，做了一桌子菜，有烤肉、凉拌菜、还有他最爱的海鲜粥🥩🍖，一家人吃得很开心！#家庭聚餐 #美食

✅ 腊月小九写法：
> 这小子两周才回来一次，这不，2026年院子里第一场烧烤就给他安排上了！#烤肉 #美食日常 #凉拌菜 @王皓琦

**差异分析**：
- 去掉Emoji / 去掉"一家人吃得很开心"的空洞表达
- 加入时间感（两周才回来一次）制造稀缺感
- 加入仪式感（"第一场"而不是"一场"）
- @王皓琦 触发儿子互动

### 示例2：复刻热点食物

❌ 普通写法：
> 今天复刻了最近超火的刘文祥麻辣，按照网上的配方自己做了一下，感觉味道差不多！大家有兴趣可以收藏试试看～

✅ 腊月小九写法：
> 我做出来了，尝尝是不是这个味 #麻辣烫 @王皓琦

**差异分析**：
- 极简正文（10字），不解释过程
- "尝尝是不是这个味"——挑战式邀请，不是推荐式劝说
- 去掉收藏引导
- 标签只用最精准的一个

---

### 示例3：投喂住宿生

❌ 普通写法：
> 儿子住校了，今天特地给他准备了可以冷吃的爆浆冰面包，步骤如下：1.和面 2.发酵 3.整形 4.冷冻 大家也可以为宿舍里的孩子做一做哦！

✅ 腊月小九写法：
> 儿子高中住校怕他早餐吃不好，给他做几个可以冷吃的冰面包当早餐吧！#爆浆冰面包 @王皓琦

**差异分析**：
- 不写步骤，只写动机（"怕他早餐吃不好"）
- "给他做"而不是"教大家做"——第一人称行动，不是第二人称建议
- 去掉所有号召语

## 第八章：局限性 + 自检清单

### 局限性说明

- **样本量限制**：本指南基于30条笔记，月更博主30条约等于2.5年内容，样本代表性较高，但近期风格可能有演变
- **视频分析缺失**：小红书限制导致约35%的笔记无法获取正文，部分分析基于标题和热评推断
- **"儿子"依赖性**：王皓琦成长（即将高考/上大学）将深刻改变账号情感结构，本指南反映当前阶段特征

### 创作自检清单

发布前逐项确认：

- [ ] 正文字数在30-80字之间（不超过100字）
- [ ] 零Emoji
- [ ] 没有操作步骤列表
- [ ] 有情感触发点（不只是食物本身）
- [ ] 标题有情感张力或疑问/挑战感
- [ ] 标签2-5个，精准垂类
- [ ] 如涉及儿子，已@王皓琦
- [ ] 没有"建议大家""值得收藏""大家可以试试"等推荐语气
- [ ] 没有白话教程感（不写"首先""然后""最后"）
- [ ] 开头有时间感或动机感，不是"今天做了XX"的平铺直叙
